For this application, transactions will be stored and manipulated using a dictionary and lists in python.
JSON file contains a dictionary with types of expenses as keys and lists of expenses as values. 
List contains amount and date as keys and their values as values:

 Amount: Float value converted into String value with two decimal points representing the transaction amount.
 Category: String representing the category of the transaction.
 Date: String representing the date of the transaction (formatted as 'yyyy-mm-dd').
